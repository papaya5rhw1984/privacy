# 개인정보처리방침 — 공통(type별) 정책  ⛔앱별 생성 금지

> 2026-06-21 CEO 확정: **앱마다 따로 만들지 않는다. type별 공통본 1개씩만 두고 그 URL을 가져다 쓴다.**

## 🚫 절대 규칙
- 새 앱 출시 시 `privacy\<앱>.html` 만들지 말 것. type 판별 후 **공통 URL을 등록정보에 그대로 입력**만 한다.
- type1/2/3 공통본은 **빈칸 채울 것 없는 완성본**(과거엔 type2·3가 [ ] 채우는 템플릿이라 앱마다 만들게 됐던 것 → 2026-06-21 완성 공통본으로 교체). 복제·수정 불필요.
- "이 앱은 좀 다른데?" 싶어도 공통본이 외부 API 조회·기기 내 저장을 포괄하므로 그대로 사용.
